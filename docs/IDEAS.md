# Ideas board

Drop ideas here before event day. Keep it short — 3 minutes to fill in, 30 seconds to read.

## Template — copy this

```markdown
### 💡 `your-name-short-slug`

**What:** _(one sentence)_

**Demo moment:** _(the 5-second on-stage action that makes judges look up)_

**Sponsors:** OpenAI / Anthropic / Fal / Gradium / Slng — circle which, one line each on how they're used.

**Gut check:**
- Heart 🩷 _(1–5, does it move people?)_
- Build 🔧 _(1–5, can we ship in 7h with our scaffold?)_
- Demo 🎬 _(1–5, wow-able in 90s?)_

**Risks:** _(one line — what kills this idea?)_
```

That's it. Six fields. If you can't fill three, the idea isn't cooked yet — leave them blank and we'll talk it through.

---

## Example

### 💡 `example-pitch-mirror`

**What:** Record your hackathon pitch, get instant feedback from 3 AI judge personas (skeptical VC, friendly mentor, tech judge).

**Demo moment:** Live-pitch on stage for 30s; judges' critiques scroll in 5s later.

**Sponsors:**
- Gradium — live STT during the pitch
- Anthropic — three persona critiques in parallel
- OpenAI — TTS the personas' voices back
- Fal — generate the "mood timeline" visual

**Gut check:** Heart 3 · Build 4 · Demo 5

**Risks:** Live STT latency on bad WiFi kills the "instant" feel.
